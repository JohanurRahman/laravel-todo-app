# Todo application with Laravel

#### This a basic Todo app made using Laravel 5.7 with CRUD functionalties

## Project setup

#### Install dependencies
```
composer install

npm install

php artisan migrate

php artisan key:generate 
```
#### Run the server
```
php artisan serve
```
